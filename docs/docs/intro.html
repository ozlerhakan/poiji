Apache Poi Version

Poiji uses Apache Poi 5.1.0.

Version

Please check the latest version on the GitHub release page. Poiji versioning is based on MAJOR.MINOR.PATCH.

All Versions

See the latest and earlier development versions including Javadoc and source files on Sonatypes OSS repository.
